Sidebar Navigation (Demo)

대시보드에서 사용한 .side-link / .nav-title 패턴. active 상태는 .active 클래스로 표현합니다.

Menu

Actions
Back to Portfolio
How to use
링크 1개 = a.side-link 현재 페이지 표시 = .side-link.active 섹션 라벨 = .nav-title
Active rule
페이지마다 해당 메뉴에만 active 클래스를 붙이면 됩니다. (JS 없어도 됨)
Example Markup
<div class="nav-title">Menu</div>

<nav class="d-grid gap-1">

  </a>

  <a class="side-link" href="#">

  </a>
</nav>
