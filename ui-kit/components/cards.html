Task Card (.card-task)

Kanban 보드에서 사용하는 기본 카드. 제목/설명/상태 dot/메타정보(날짜, 서브태스크 등)로 구성.

Design Login UI

Design • Glass + Neon

Implement Todo CRUD

Feature • API 연결

Example Markup
<article class="card-task">
  <div class="task-top">
    <div>
      <p class="task-name">Design Login UI</p>
      <p class="task-desc">Design • Glass + Neon</p>
    </div>
    <span class="dot yellow"></span>
  </div>
  <div class="task-foot">
    <span><i class="bi bi-calendar2"></i> 02/10</span>
    <span><i class="bi bi-list-check"></i> 2 subtasks</span>
  </div>
</article>

Utility Card (Documentation)

UI Kit 문서 페이지에서 활용 가능한 “설명 카드” 스타일 예시.

Usage Tip
task 카드 자체는 dashboard.css에 정의되어 있어도 괜찮습니다. 다만 재사용도가 높아지면 components.css로 승격시키는 게 유지보수에 유리합니다.
