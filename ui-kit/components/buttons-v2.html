Split Button

Usage

<!-- soft / ghost --> <button class="btn btn-soft rounded-pill">Soft</button> <button class="btn btn-ghost rounded-pill">Ghost</button> <!-- icon only --> <button class="btn btn-soft btn-icon" aria-label="add"> <i class="bi bi-plus-lg"></i> </button> <!-- split button --> <div class="btn-split"> <button class="main" type="button">Export</button> <button class="drop" type="button" aria-label="open menu"> <i class="bi bi-chevron-down"></i> </button> </div>

Guidelines

  • Primary CTA는 .btn-brand 1개만(한 화면에 여러 개 남발 금지)
  • Icon-only는 aria-label 필수, 최소 터치 영역 32px 권장
  • Split button 드롭은 실제 구현 시 dropdown 메뉴로 연결
