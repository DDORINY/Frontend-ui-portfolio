Topbar v2

pill topbar + search + icon actions
Back Home
v2 Changes
  • Topbar를 pill 형태로 표준화(내부 페이지 공통)
  • 검색창(아이콘 포함) + 우측 액션 아이콘 그룹

Demo

DDORINY Dashboard

Usage

<div class="topbar-v2 d-flex align-items-center gap-2 flex-wrap"> <div class="brand">...</div> <div class="search"> <i class="bi bi-search"></i> <input type="text" placeholder="Search..."> </div> <div class="d-flex align-items-center gap-2 ms-auto"> <button class="btn btn-ghost btn-icon btn-sm">...</button> <button class="btn btn-soft rounded-pill">New</button> </div> </div>

Guidelines

  • Topbar는 내부 페이지 공통(상단) 배치
  • 검색은 폭이 줄면 자동 줄바꿈되도록 flex-wrap 유지
  • 우측 액션은 아이콘 버튼 + 주요 CTA 1개만 유지
