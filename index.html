Usage

정적 HTML 기반입니다. index.html을 브라우저로 열어 전체 프리뷰를 확인하세요.
  • theme.css : 디자인 토큰(변수)
  • components.css : 재사용 컴포넌트
  • pages/*.css : 페이지 전용 스타일
  • ui-kit/ : 문서(규칙/샘플/스니펫)
